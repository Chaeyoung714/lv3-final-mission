
# 레벨3 최종 미션
# 나만의 뮤지컬 예매 서비스

## 서비스 개요
뮤지컬의 티켓을 예매할 수 있는 서비스이다.

## 아키텍처
- 운영서버, 개발서버, 운영DB의 물리/네트워크 환경이 분리되어있습니다.
- 각 단계에서 빌드 및 테스트 통과를 확인함으로써 안정적인 통합이 가능합니다.
- 배포 시 git 명령, 빌드, 배포 과정에서 실패할 경우 에러 사유를 자세히 안내합니다.
- CI/CD 플로우에서 중복되는 흐름이 없도록 하여 배포 시간을 최소화하였습니다.
- Flyway를 통해 운영서버에서는 데이터를 유지한 채 스키마를 변경할 수 있습니다.
